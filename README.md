# CHECKOUT M3 ACADEMY

## Instalação e desenvolvimento

    - abra o terminal
    - yarn install
    - cd checkout
    - npm i

## Comando para rodar o projeto

    - na pasta checkout (cd checkout assim que abrir o projeto)
    - rodar: npm run dev
    
## Versão do node.js:

    - NODEJS >= 12
